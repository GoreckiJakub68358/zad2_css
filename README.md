## 📋 Opis projektu

Strona internetowa w formacie CV wykonana w czystym HTML5 z zewnętrznymi stylami CSS (External CSS).  
Projekt realizowany w ramach zadania zaliczeniowego.

## ✅ Wymagania techniczne

- [x] Poprawne podłączenie arkusza stylów z wykorzystaniem znacznika `<link>`
- [x] Brak stylów inline (brak atrybutu `style` w znacznikach HTML)

`style.css` zawiera:
- [x] Ustawienie koloru przewodniego strony w kolorze zielonym lub jego odcieniu (kolor ma należeć do gamy zieleni i być widoczny na stronie)
- [x] Zmiana kroju pisma (`font-family` inny niż domyślny)
- [x] Ustawienie marginesów i odstępów (np. `margin` lub `padding` dla `<body>` albo sekcji)
- [x] Styl widoczny wizualnie
